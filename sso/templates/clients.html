Security Best Practices
Important: Never share your client secrets publicly. Store them in environment variables.
  • Use HTTPS for all redirect URIs in production
  • Enable PKCE for public clients (SPAs, mobile apps)
  • Rotate client secrets periodically (every 90 days)
  • Use appropriate scopes (minimum required permissions)
  • Implement CORS properly for web applications
  • Monitor and audit client usage regularly
Client Types:
Confidential Web servers, daemons, APIs (can keep secrets)
Public SPAs, mobile apps, desktop apps (cannot keep secrets)
